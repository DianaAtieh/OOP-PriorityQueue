# OOP Priority Queue (Java)

## 📌 Description
This project is an implementation of a Priority Queue using Object-Oriented Programming (OOP) concepts in Java.

The system simulates a simple printing server where print jobs are handled based on priority.

---

## 🚀 Features
- Custom Priority Queue implementation
- Use of interfaces (`Prioritizable`)
- Classes and OOP design
- Insert and remove operations based on priority
- Simple simulation of a printing system

---

## 🧠 Concepts Used
- OOP (Classes, Objects, Interfaces)
- Encapsulation
- Arrays
- Basic data structures

---

## 📂 Project Structure
- `Prioritizable.java` → Interface for priority handling
- `PriorityQueue.java` → Queue implementation
- `PrintJob.java` → Represents a job
- `PrintingServer.java` → Manages jobs
- `Tester.java` → Test and run the system

---

## ▶️ How to Run
1. Open the project in IntelliJ
2. Run `Tester.java`
